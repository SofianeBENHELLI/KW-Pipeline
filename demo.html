Two-step launch

1
Start the backend
Boots uvicorn on :8000. First run creates a Python venv and installs apps/api[test] (~30 s); subsequent runs are ~2 s.
Terminal 
./scripts/demo-backend.sh
…or in Finder double-click Demo Backend.command at the repo root
2
Start the frontend
Boots Vite on :5174 mounting the real widget. First run runs npm install (~20 s); subsequent runs are ~1 s.
Terminal 
./scripts/demo-frontend.sh
…or in Finder double-click Demo Frontend.command at the repo root
