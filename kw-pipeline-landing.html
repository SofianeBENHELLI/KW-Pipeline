Lancer en local

Le backend tourne sur :8000, Orbital sur :5173. Choisis une des deux méthodes ci-dessous.

Méthode 1 — App launcher (Safari recommandé)

▶ Lancer le backend (API) ▶ Lancer le frontend (Orbital)

La première fois, macOS bloque l'app non signée : clic-droit sur le .app dans Finder → Ouvrir → Ouvrir (une seule fois). Ensuite le bouton fonctionne.

Méthode 2 — Copier la commande puis coller dans Terminal

Ouvre Terminal (Cmd+Space → "Terminal") et colle (Cmd+V) puis Entrée.

Méthode 3 — Commande npm directe (fallback)

Si les méthodes ci-dessus échouent : colle dans Terminal et lance le serveur Vite à la main. L'URL http://localhost:5173/ sera disponible quand tu vois VITE ready.
